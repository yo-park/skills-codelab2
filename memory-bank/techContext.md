# Tech Context

## Technologies Used
- **Backend**: Rust
- **Frontend**: React, TypeScript, Node.js
- **Other**: Git, macOS (OS)

## Development Setup
- The project structure includes separate directories for `app_build/backend` and `app_build/frontend`.
- Development workflow involves working with both Rust and TypeScript environments.

## Technical Constraints
- The project is designed to run on a local machine with necessary runtimes installed (Rust/Cargo, Node.js).
- Documentation is maintained in a dedicated `memory-bank/` directory.

## Dependencies
- Rust toolchain
- Node.js and npm/yarn
- TypeScript
